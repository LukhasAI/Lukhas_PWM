### Quick Start

```bash

git clone https://github.com/LukhasAI/Lukhas_PWM.git
cd Lukhas_PWM

# Install dependencies
pip install -r requirements.txt

# Initialize the system
python main.py --setup

# Start the PWM interface
python main.py --start
```
